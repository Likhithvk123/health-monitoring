Health Monitoring Using Machine Learning

June-1 Team 10 Project

In this Health monitoring system we will be detecting the level of ill health of the person and would recommend few medications that can be taken by him by which he can recover a bit than before and here we are seeing some parameters of the person which are blood pressure,pulse and temperature by these parameters we can analyze his health status from 0-2 where 0 is that he is perfectly okay, 2 is that the person is very much ill that he need to consult doctor.
